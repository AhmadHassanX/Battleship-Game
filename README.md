# Battleship-Game
A fun console-based Battleship game in C++ where you destroy hidden ships using limited cannonballs. Features color-coded map visualization and a structured single-player gameplay experience.

## Game Modes
- **Single Player:** Hunt down 5 randomly placed enemy ships using only 10 shots.
- **VS Computer (Coming Soon):** A planned mode where you‚Äôll battle an AI opponent.
- **Multiplayer (Coming Soon):** A future local two-player implementation.
- **How to Play:** In-game guide explaining symbols, controls, and objective.
  
## Features
- 5x5 grid with randomly placed ships.
- Colored output for hits (`X`), misses (`O`), and remaining boats (`B`).
- Input validation for coordinates.
- Score tracking and win/loss message.
- Clean interface using `conio.h` and `system("cls")`.

## Tools & Libraries
- **Language:** C++
- **IDE:** Visual Studio Code
- **Compiler:** Typically uses MinGW or MSVC toolchain integrated with VS Code on Windows
- **Libraries Used:**
  - `<iostream>` for input/output operations
  - `<cstdlib>` and `<ctime>` for random number generation
  - `<conio.h>` for console input handling (Windows only)
